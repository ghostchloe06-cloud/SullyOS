🕵️ 角色卡危险信息检测 本地测试版 · 不接入正式版

拖入 / 选择 / 粘贴一张角色卡 JSON，检测里面是否带了 API 密钥等危险信息。全程在本地浏览器运行，不上传任何数据。
把 角色卡 .json 拖到这里，或 点击选择文件
检测逻辑：
1. 深度递归扫描整份 JSON——任何字段名像密钥（apiKey / secret / token / authorization / bearer / password / anonKey …）或字段值像密钥（sk-…、Bearer …、JWT、长十六进制/base64）的，一律标红。将来新增字段也能覆盖。
2. 按已知清单核对美化 / 语言 / 运行时状态字段是否存在（导入正式版时这些会被剥离，但老卡文件本身仍可能带着）。
密钥值默认打码显示，只露头尾几位供你比对。
